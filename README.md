## Пет-проект для мониторинга системных метрик на Python
Что умеет:
- Находит системные характеристики Linux системы (Модель Процессора, Объем ОЗУ, модель GPU на базе Nvidia)
- Парсить на заданном интервале времени температуру устройств материнской платы (Тесты показали Ethernet порт, Процессор, NVME M2 SSD)
- На основе собранных метрик строить графики с сохранением в формате png
- Обрабатывать ошибки и логиравоть на трех уровнях: INFO, WARNING, ERROR с выявлением проблемного модуля и записью даты ошибки
